# My First Website Layout

## 📌 Overview
This project is a foundational web page layout built to demonstrate proper document structure and modern web standards. 

## 🧠 My Approach
My primary goal for this project was to create a clean, accessible, and well-organized page structure. Instead of relying on generic `<div>` tags, I utilized **HTML5 Semantic Layout Tags**. 

By using tags like `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>`, the code is:
* **Accessible:** Screen readers and assistive technologies can easily navigate the different areas of the page.
* **Readable:** The code is logically divided, making it easier for other developers to read and maintain.
* **SEO Friendly:** Search engines can better understand the hierarchy and importance of the content.

## 💻 Technologies Used
* **HTML5:** For the core structure and semantic markup.
* **GitHub Pages:** For live hosting and deployment.
